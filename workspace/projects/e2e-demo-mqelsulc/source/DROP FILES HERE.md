# Drop your project material here

Put whatever describes your product into these folders, then run the pipeline.

- **repo-clone/** — clone or copy your code repo here (`git clone <url> repo-clone`).
- **notes/** — any .md / .txt notes, briefs, feature lists, positioning.
- **raw-html/** — existing pages, exports, or HTML you want considered.

Jarvis reads the text from these folders to understand the project.
